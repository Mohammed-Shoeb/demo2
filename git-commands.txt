CLI Vendors :

GIT : GNU(Open Source) | CLI
Git is a distributed revision control system with an emphasis on speed data integrity and support for distrubuted non-linear workflows. 
Git was initially designed and developed by Linus Torvalds for linux kernel development in 2005 and has since become the most widely adopted version control system for software development

https://git-scm.com/ 

GUI Vendors :

GITHUB : GUI | Public & Private Repository Vendors :
GitHub is a web-based Git repository hosting service which offers all of the distributed revision control and source code management (SCM) functionally of Git as well as adding its own features Unlike Git which is strictly a command-line tool GitHub provides a web-based graphical interface and desktop as well as mobile integration

GITLAB SCM : | Public & Private Repository Vendors :
GitLab's SCM (Source code management) solution supports software development teams to collaborate communicate in order to quickly solve problems and deliver new features.

ATLASSIAN BITBUCKET | Public & Private Repository Vendors : 
Bitbucket is a web-based hosting service for projects that use either the Mercurial (since launch) or Git (since launch) or Git (since October 2011[1]) revision control systems Bitbucket offers both commercial plans and free accounts.
